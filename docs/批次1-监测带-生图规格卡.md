## 共同硬规则（每张都要）

1. **背景一律纯绿幕**：物件卡（②-⑥）的背景必须是**一整片纯绿色**（提示词已写好 solid pure green #00FF00）。你的工具出不了透明 PNG，绿幕是这个项目一直在用的老办法（我负责抠绿）。**别要求透明底，别用棋盘格底**。
2. **工具里选比例，分辨率不用管**：每张卡写了"工具里选什么比例"。生成出来是 1024 还是 2048 都行，**最终尺寸我来缩**。
3. **30 秒像素粗筛**：出图后放大到 400% 看边缘——合格 = 一格一格的硬台阶；不合格 = 模糊渐变、毛玻璃边。**拿不准就直接给我，我跑行程统计脚本验**，不合格我会指出重 roll 哪张。
4. **物件卡（②-⑥）统一三查**：① 背景是一整片纯绿、绿色没混进物体本体 ② 画面里只有这一个物件——没有地面、没有影子 ③ 400% 看边缘是硬台阶。
5. 整体冷蓝灰调；**全批次禁琥珀禁暖棕**（琥珀是凤翎专属的全画面唯一暖色）；病态黄绿也禁用（监测带还没到异常区）。建筑灯光一律青绿 #6fd3c4。

## 生成顺序与风格链（一处定义，照这个做）

1. **① 地貌**先做，调到满意 → 它是这批的风格基准
2. **② ④ ⑤ ⑥ 都用 ① 当风格参考**；**③ 用 ② 当参考**（同族建筑优先于地貌）
3. 用工具的**风格参考 / sref 模式**，别用垫图/重绘模式
4. ⚠️ 参考图只学色调——如果生成结果里**长出了地面**或**整体发绿**，说明参考权重太高，调低或去掉参考重 roll
